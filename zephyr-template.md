# Zephyr CSV Template Guide

## Required Columns

The CSV file must include the following columns in order:

1. Test Case Name
2. Objective
3. Precondition
4. Status
5. Test Step
6. Expected Result
7. Priority
8. Label

## Formatting Requirements

- Ensure each test case field maps to its corresponding CSV column
- Escape special characters properly (e.g., wrap fields containing commas or quotes in double quotes, and escape internal quotes by doubling them)
